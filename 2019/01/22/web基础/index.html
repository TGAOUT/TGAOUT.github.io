windows历史版本

Windows主要有桌面版和服务器版、移动版三个版本

  1. 桌面版现在主流是WindowsXP、WindowsVista、Windows7、Windows8、Windows10
  2. 服务器版主流:WindowsServer2003、WindowsServer2008、WindowsServer2012、WindowsServer2016
  3. 移动版主要有WindowsMobile、WindowsPhone、Windows10Mobile
  4. 另外还有一个嵌入系统WindowsCE，一般是工业上用的

什么是服务器？

服务器可以理解为电脑，不过只是比家用电脑性能牛逼多了。

IP基础知识

什么是IP地址？
大千世界中，每一台得IP地址就相当于我们得身份证ID

  • A类IP: 10.0.0.0~10.255.255.255 即10.0.0.0/8
  • B类IP: 172.16.0.0~172.31.255.255即172.16.0.0/12
  • C类IP: 192.168.0.0~192.168.255.255 即192.168.0.0/16
  • 本机IP: 127.0.0.1或 localhost
  • 公网
  • 广域网
  • 局域网IP 连接到同一台路由器下面得电脑所组成得就叫做局域网

端口知识

共有65535个端口。
可以理解为，每个端口对应一个服务。

服务 默认端口
http 80
https 443
ftp 21
mysql 3306
Oracle 1521
Mssql 1433
远程桌面服务端口 3359
Tomcat 8080

另外，https在通信时有个加解密过程

常见状态码

状态码 涵义
200 存在某个文件
301 永久跳转(应用场景如更换网址)
302 临时跳转(应用场景如登录)
304 可以理解为从本地读取的网页缓存
403 无权限访问目录或文件
404 目录或文件不存在
500 系统错误或代码错误
502 网关错误

什么是CDN？

主要作用：在cdn服务器上缓存一些文件，缓解网站服务器的压力，提高访问速度。
